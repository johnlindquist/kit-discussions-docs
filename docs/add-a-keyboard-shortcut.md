Use the `Shortcut` metadata to add a global keyboard shortcut to any script

```js
// Shortcut: cmd shift j

import "@johnlindquist/kit"

say(`You pressed command shift j`)
```

```js
// Shortcut: opt i

import "@johnlindquist/kit"

say(`You pressed option i`)
```
